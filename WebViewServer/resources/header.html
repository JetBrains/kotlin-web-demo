Web Demo runs Kotlin type checker to report errors/warnings in the editor. The type checker can run either on your machine (using a Java™ Applet) or on our server (this is slower).

Client mode (recommended): type checker, code completion and other services are running on your machine (in a Java™ Applet). The server only generates byte code and runs your programs on the JVM.

Server mode: all the services except for basic keyword highlighting are running on our server.

No highlighting mode (for slow connection): type checker is running on our server only when you run your program.
